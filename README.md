Nome do repositório: BancoJava
Descrição curta:

Sistema bancário simples em Java com interface gráfica (GUI) usando Swing.

Descrição detalhada:

Este projeto consiste em um sistema bancário básico desenvolvido em Java. Ele permite:

Criar uma conta com nome do usuário e saldo inicial.

Realizar depósitos e levantamentos com validação de saldo.

Consultar extrato da conta, mostrando nome, número da conta, saldo e número de levantamentos.

Autenticação de usuário com login e senha antes de acessar o menu principal.

Interface gráfica desenvolvida com Java Swing, organizada em múltiplas telas (autenticação e menu de operações).

Objetivos do projeto:

Praticar Programação Orientada a Objetos em Java (classes, encapsulamento, métodos e atributos).

Desenvolver habilidades em GUI com Swing.

Implementar lógica de operações bancárias simples.

Estrutura do projeto:

Teste1/Conta.java → Classe que representa uma conta bancária.

Teste1/Menu.java → Tela de menu com opções de operação (depósito, levantamento, extrato).

Teste1/TelaAutenticacao.java → Tela de login/autenticação do usuário.
